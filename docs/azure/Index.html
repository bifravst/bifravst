Bifravst on Azure¶

Note

The Azure implementation is a work in progress.

Bifravst on Azure is a reference implementation of a serverless backend for an IoT product developed using Azure Resource Manager in TypeScript.

Note

The Azure solution is using Cosmos DB for querying historical device data and hence the minimum costs to run Bifravst on Azure is around $24 per month. However, you can check if you are eligible for the free tier for new accounts.
