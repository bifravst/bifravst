================================================================================
Bifravst on AWS
================================================================================

Bifravst on AWS is a reference implementation of a serverless backend for an IoT
product developed using `AWS CDK <https://aws.amazon.com/cdk/>`_ in
`TypeScript <https://www.typescriptlang.org/>`_.
